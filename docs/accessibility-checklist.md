# Accessibility Checklist

- Landmarks present: banner, main, contentinfo.
- Headings follow logical order (`h1` per page, then descending).
- All controls reachable via keyboard.
- Focus styles visible using `.focus-ring`.
- No critical info exposed only on hover.
- Text contrast reviewed against WCAG AA.
- All meaningful images include alt text.
- `prefers-reduced-motion` respected in global styles.
- Map has fallback: direct maps link and text directions.
